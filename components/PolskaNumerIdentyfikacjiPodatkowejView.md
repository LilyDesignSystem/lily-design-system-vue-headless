# PolskaNumerIdentyfikacjiPodatkowejView

A headless Vue 3 read-only display for Poland's Numer Identyfikacji Podatkowej (NIP).

Format: 10 numeric digits used for tax identification.

Companion: `PolskaNumerIdentyfikacjiPodatkowejInput`.

References: https://en.wikipedia.org/wiki/National_identification_number#Poland
