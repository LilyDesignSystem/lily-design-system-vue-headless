# TableFoot

The footer section of a Table, rendered as a <tfoot> element. Contains TableRow elements with summary or footer cells.

## Props

- default slot — content rendered inside the `<tfoot>`

## Usage

```vue
<TableFoot>
  ...
</TableFoot>
```
