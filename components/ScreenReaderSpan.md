# ScreenReaderSpan

A visually hidden span of text intended for screen readers.

## Props

- `label`: string (optional) -- accessible label via `aria-label`
- default slot — component content

## Usage

```vue
<ScreenReaderSpan>...</ScreenReaderSpan>
```

## References
